# NeuroSymbolic-CADD
Autonomous local neuro-symbolic system for computer-aided drug design (CADD).
This project combines the flexibility of Large Language Models (LLMs) with deterministic chemical algorithms (RDKit, AutoDock Vina, D-MPNN).

Developed as a Master's Thesis at Taras Shevchenko National University of Kyiv (Institute of High Technologies).
